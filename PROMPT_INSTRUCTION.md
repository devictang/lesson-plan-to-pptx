# 🤖 格式附錄（Format Appendix）

以下係一個**格式控制附錄**。使用方式：

1. 先同 AI 討論你嘅教案內容（例如「幫我設計一個中三生物科光合作用嘅教案」）
2. 將下面嘅格式附錄 **append** 喺你嘅 instruction 最後
3. AI 就會輸出符合 parser 嘅格式，直接貼上網站即可生成 PowerPoint

---

## 📋 複製以下內容，append 喺你嘅 instruction 最後：

```
**輸出格式要求（必須嚴格遵守）：**

1. 第一行使用 `# 主題名稱`
2. 每個 `## ` 開頭代表一張新 Slide
3. 第一個 `## ` 固定為封面 Slide

**7 種 Slide 語法：**

| 類型 | 語法 | 效果 |
|------|------|------|
| 封面 | 第一個 `## 封面 — 主題` + `- **key**：value` | 深色全幅標題 |
| 分隔頁 | `## ---` + section 名稱 | 全色背景置中大字 |
| 引用頁 | `> 引文` + `> — 署名` | 大引號 + 斜體金句 |
| 統計頁 | `- **標籤**：數值` (1–4 組) | 超大數字 + label |
| 表格頁 | `| A | B |` + `|---|---|` | 結構化表格 |
| 對照頁 | `## 左標題 || 右標題` + 內容用 `---` 分隔 | 雙欄對比 |
| 內容頁 | `1. ` / `- ` 列表 或 段落文字 | 一般內容 |

**完整範例：**

```
# 教學主題：[主題名稱]

## 封面 — [主題名稱]
- **科目**：[科目]
- **年級**：[年級]
- **時間**：[分鐘] 分鐘

## 教學目標
1. [目標一]
2. [目標二]

## ---
核心教學

## 光合作用方程式
| 原料 | 條件 | 產物 |
|------|------|------|
| CO₂ + H₂O | 光 + 葉綠體 | C₆H₁₂O₆ + O₂

## 導入活動 || 總結活動
- 提問引導
- 影片播放
---
- 完成工作紙
- 教師總結

## 光合作用的重要性
> 植物係地球上最重要嘅生產者。
> — 生物學教科書

## ---
課堂數據

## 本課重點
- **課時**：40 分鐘
- **實驗次數**：3
```

**注意：**
- 分隔頁用 `## ---` + section 名切換章節
- 統計頁只放 key-value，唔好加其他文字
- 表格必須有 `|---|---|` separator row
- 對照頁用 `||` 分標題，內容用 `---` 分隔左右欄
- 引用用 `> ` 開頭，`> — Name` 係署名
```

## 🎯 7 種 Slide 類型速查

| # | 類型 | 觸發條件 | 視覺效果 |
|---|------|----------|----------|
| 1 | 封面 | 第一個 `## ` | 全 accent 色背景 + 大標題 + meta info |
| 2 | 分隔頁 | `## ---` | 全 accent 背景 + 置中大字（無 slide number） |
| 3 | 引用頁 | `> ` 開頭 | 超大引號 + 斜體金句 + 右對齊署名 |
| 4 | 統計頁 | 1–4 組 key-value | 1 組置中 / 多組並排 + vertical divider |
| 5 | 表格頁 | `|...|` pipe table | Accent header + 交替 row colors |
| 6 | 對照頁 | `||` in title | 雙欄 + vertical divider，內容用 `---` 分左右 |
| 7 | 內容頁 | 預設 | 列表（number circle / bullet dot）/ 段落文字 · 短內容有 half-bleed accent panel |

## ⚠️ 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 將格式附錄當作 standalone prompt | 必須 append 喺你自己嘅 instruction 最後 |
| 分隔頁冇 section 名 | `## ---` 後面寫 section 名稱 |
| 統計頁混入其他文字 | 只放 `- **label**：value` |
| 表格冇 separator row | 必須有 `\|---\|---\|` |
| 對照頁冇 `\|\|` 分標題 | `## 左 || 右` |
| 對照頁內容冇 `---` 分隔 | 左右欄內容用 `---` 分開 |
