# 🤖 格式附錄（Format Appendix）

以下係一個**格式控制附錄**。使用方式：

1. 先同 AI 討論你嘅教案內容（例如「幫我設計一個中三生物科光合作用嘅教案」）
2. 將下面嘅格式附錄 **append** 喺你嘅 instruction 最後
3. AI 就會輸出符合 parser 嘅格式，直接貼上網站即可生成 PowerPoint

---

## 📋 複製以下內容，append 喺你嘅 instruction 最後：

```
**輸出格式要求（必須嚴格遵守）：**

1. 第一行使用 `# 主題名稱`
2. 每個 `## ` 開頭代表一張新 Slide
3. 第一個 `## ` 固定為封面 Slide

**12 種 Slide 語法：**

| 類型 | 語法 | 效果 |
|------|------|------|
| 封面 | 第一個 `## 封面 — 主題` + `- **key**：value` | 深色全幅標題 + meta info |
| 分隔頁 | `## ---` + section 名稱 | 全色背景置中大字 |
| 引用頁 | `> 引文` + `> — 署名` | 大引號 + 斜體金句 |
| 統計頁 | `- **標籤**：數值` (1–4 組) | 超大數字 + label |
| 表格頁 | `| A \| B \|` + `|---|---|` | 結構化表格 |
| 對照頁 | `## 左標題 \|\| 右標題` + 內容用 `---` 分隔 | 雙欄對比 + vertical divider |
| **概念卡** | `> **概念名**：說明文字` (2-3 組) | 多個 rounded card 並排 |
| **提示框** | `> 💡 提示文字` | 左 accent border 嘅 callout box |
| **練習區** | `> ✏️ 練習標題` + 內容 | 邊框 box + 「練習」label |
| **選擇題** | `> ❓ 題目` + 選項 `A. xxx`  + `✅ 正確答案` | Circle letter + 綠色標記正確 |
| **程式碼** | ` ```語言` + code + ` ``` ` | 暗色背景 + monospace code block |
| **時間線** | slide title 有「時間線/roadmap」+ 列表 | 橫向時間線 timeline |
| 內容頁 | `1. ` / `- ` 列表 或 段落文字 | 一般內容（有 learning objectives sidebar） |

**📝 逐字稿（Speaker Notes）：**
- 喺任何 slide 內容最後加 `> notes：` 開頭
- 跟住嘅文字會存入 PPTX 嘅 speaker notes，只 presenter 睇到
- 遵守三鐵律：提示信號格式 / 150-300 字 / 口語書寫

**完整範例：**

```
# 教學主題：光合作用

## 封面 — 光合作用
- **科目**：生物科
- **年級**：中三
- **時間**：40 分鐘

## 教學目標
1. 說明光合作用的定義及化學方程式
2. 辨識光合作用所需的條件與產物
3. 理解光合作用對生態系統的重要性

## ---
認識光合作用

## 光合作用方程式
| 原料 | 條件 | 產物 |
|------|------|------|
| 二氧化碳 + 水 | 光 + 葉綠體 | 葡萄糖 + 氧氣 |
| 6CO₂ + 6H₂O | 光合作用 | C₆H₁₂O₆ + 6O₂

## 光合作用嘅核心概念
> **原料**：二氧化碳 + 水，由葉片氣孔吸收
> **條件**：陽光 + 葉綠體，缺一不可
> **產物**：葡萄糖 + 氧氣，為生態系提供能量
> notes：呢度用三張概念卡解釋光合作用嘅三個要素。先講原料由邊度嚟——葉片氣孔吸收 CO₂，根部吸水。再講條件——陽光係能量來源，葉綠體係工廠。最後講產物——葡萄糖俾植物用，氧氣俾我哋用。

## 💡 常見誤解
> 💡 好多學生以為光合作用係響夜晚進行。實際上，光合作用必須有光！夜晚植物只進行呼吸作用。

## ✏️ 小組練習
> ✏️ 平衡方程式練習
> 1. 寫出光合作用嘅完整化學方程式
> 2. 標明每個物質嘅狀態（固/液/氣）
> 3. 解釋如果 CO₂ 濃度增加一倍會點樣
> notes：呢個練習大約 5 分鐘。學生可以兩人一組討論。第一條比較直接，第二條要留意水係液態。第三題係挑戰題——同學生講 CO₂ 增加理論上會加速光合作用，但其他限制因素（光、溫度）都會影響。

## ❓ 課堂測驗
> ❓ 以下邊個唔係光合作用所需嘅條件？
> A. 陽光
> B. 水
> C. 氧氣
> D. 葉綠體
> ✅ C

## 「光合作用的重要性」引入||「光合作用的重要性」總結
- 植物係地球上嘅生產者
- 提供氧氣俾生物呼吸
- 減少大氣中嘅二氧化碳
---
- 完成工作紙嘅題目 1-5
- 每組分享一個生活例子

## ---
本課數據

## 本課重點
- **課時**：40 分鐘
- **實驗次數**：3
- **小組數量**：4

> notes：總結今日重點。光合作用嘅方程式要記熟，下一堂會講呼吸作用嘅對比。
```

**注意：**
- 分隔頁用 `## ---` + section 名切換章節
- 統計頁只放 key-value，唔好加其他文字
- 表格必須有 `|---|---|` separator row
- 對照頁用 `||` 分標題，內容用 `---` 分隔左右欄
- 引用用 `> ` 開頭，`> — Name` 係署名
- **概念卡**：連續 2-3 行 `> **概念**：說明` 自動變卡片
- **提示框**：用 `> 💡` 開頭
- **練習區**：用 `> ✏️` 開頭
- **選擇題**：用 `> ❓` 開頭，`✅` 標記正確答案
- **程式碼**：用 ` ``` ` 包圍
- **逐字稿**：用 `> notes：` 開頭，每頁最後加
```

## 🎨 主題推薦（12 款，按教學場景）

| 教學場景 | 推薦主題 | 風格 |
|---------|---------|------|
| **大學/論文答辯** | `academic-paper` | 嚴肅 serif、零圓角 |
| **教案/線上課程** ⭐ | `course-module` | 暖米色、學習目標 sidebar |
| **中小學常規課** | `corporate-clean` | 專業藍色、乾淨俐落 |
| **語文/文學/歷史** | `editorial-serif` | 優雅 serif、深紅 accent |
| **數學/科學/編程** | `tech-sharing` | GitHub dark、code-friendly |
| **生物/地理/環境** | `nature-green` | 自然綠色、平和有機 |
| **物理/工程** | `blueprint` | 深藍工程圖、金黃 accent |
| **化學/醫學** | `ocean-blue` | 冷藍清晰、冷靜專業 |
| **歷史/社會學** | `history-amber` | 羊皮紙懷舊、磚紅 accent |
| **小學/幼教** | `pastel-warm` | 大圓角、柔和色彩 |
| **資料/統計課** | `swiss-grid` | 紅色 accent、強烈網格 |
| **設計/藝術課** | `minimal-white` | 極簡留白、黑白配色 |

## ⚠️ 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 將格式附錄當作 standalone prompt | 必須 append 喺你自己嘅 instruction 最後 |
| 分隔頁冇 section 名 | `## ---` 後面寫 section 名稱 |
| 統計頁混入其他文字 | 只放 `- **label**：value` |
| 表格冇 separator row | 必須有 `\|---\|---\|` |
| 對照頁冇 `\|\|` 分標題 | `## 左 \|\| 右` |
| 對照頁內容冇 `---` 分隔 | 左右欄內容用 `---` 分開 |
| 概念卡少過 2 張 | 至少要 2 組 `> **概念**：xxx` 先會變卡片 |
| 逐字稿寫太長 | 每頁 150-300 字就夠 |
| 逐字稿似讀稿 | 用口語 + 加粗關鍵提示信號 |
