# 🤖 AI 教案生成指令

將以下指令**完整複製**，貼到 ChatGPT、Claude 或其他 AI 助手，AI 會按照每張 Slide 的結構生成教案內容。生成的內容可直接貼上網站，一鍵轉換為 PowerPoint。

---

## 📋 複製以下內容給 AI：

```
你是一個專業的教案簡報生成助手。請根據用戶提供的主題，生成一份可以直接轉換為 PowerPoint 的結構化 Markdown。

**格式規則（必須嚴格遵守）：**

1. 第一行使用 `# 主題名稱` 作為整份簡報的標題
2. 每個 `## ` 開頭的行代表一張新 Slide
3. 第一個 `## ` 是封面 Slide，可在內容中使用 `- **key**：value` 格式加入基本資訊（科目、年級、時間等）
4. 後續每個 `## ` 都是一張內容 Slide，標題後面可以寫段落文字
5. 如需顯示列表，使用 `1. ` （編號列表）或 `- ` （項目符號列表），系統會自動加上精美設計

**格式範例：**

```
# 教學主題：[主題名稱]

## 封面 — [主題名稱]
- **科目**：[科目]
- **年級**：[年級]
- **時間**：[分鐘] 分鐘

## [Slide 標題]
[內容段落…]

## [Slide 標題]
1. [列表項目一]
2. [列表項目二]
3. [列表項目三]

## [Slide 標題]
- [項目符號一]
- [項目符號二]
```

**注意事項：**
- 每個 `## ` 就是一張 Slide，請按你想要的簡報順序排列
- 不要使用 `### ` 或其他層級的標題
- 列表（`1. ` 或 `- `）會被自動美化為帶圖標的設計
- 非列表的內容會以段落文字形式顯示
- 輸出的簡報會自動在最後加上「謝謝」結尾頁
```

---

## 📝 使用方式

1. **複製** 上面框框內的完整指令
2. **貼到** ChatGPT / Claude / Gemini 等 AI 工具，並加上你的主題要求（例如：「主題是光合作用，適合中三生物科」）
3. AI 會輸出按 Slide 排列的 Markdown 內容
4. 將輸出**貼到** [教案轉 PPTX 網站](https://lesson-plan-to-pptx.vercel.app)，即可一鍵下載精美 PowerPoint

## ✅ 格式檢查清單

- [ ] 每個 Slide 以 `## ` 開頭
- [ ] 第一個 `## ` 為封面 Slide，包含基本資訊
- [ ] 列表使用 `1. `（編號）或 `- `（項目符號）

## 🌟 範例輸出

以下是 AI 應該生成的真實格式範例：

```
# 教學主題：光合作用

## 封面 — 光合作用
- **科目**：生物科
- **年級**：中三
- **時間**：40 分鐘

## 教學目標
1. 說明光合作用的定義及化學方程式
2. 辨識光合作用所需的條件與產物
3. 理解光合作用對生態系統的重要性

## 教學資源
- 教科書第四章
- 光合作用動畫影片
- 實驗器材（碘液、酒精、本生燈）
- 工作紙

## 課堂導入（5 分鐘）
播放一段植物生長的快鏡影片，提問：「植物為什麼不需要吃東西也能長大？」引導學生思考植物如何獲取能量。

## 光合作用的過程（15 分鐘）
教師講解光合作用的基本概念與化學方程式。介紹光反應和暗反應的區別，以及葉綠體的角色。

## 實驗示範（10 分鐘）
進行光合作用實驗示範，檢測葉片中澱粉的存在。

## 小組討論與總結（10 分鐘）
- 小組討論光合作用對生態系統的影響
- 學生完成工作紙
- 教師選取 2-3 位學生分享答案並給予回饋
```

## ⚠️ 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 使用 `### ` 分隔步驟 | 統一使用 `## `，每個步驟都是一張獨立 Slide |
| 封面沒有放在第一個 `## ` | 第一個 `## ` 必定是封面 Slide |
| 基本資訊格式不一致 | 統一使用 `- **key**：value` |
| 加入了多餘的 markdown 格式 | 只使用 `#`、`## `、`1. `、`- ` |
| 列表項目與段落文字混合太亂 | 如果內容以列表為主就用列表，否則用段落文字 |
