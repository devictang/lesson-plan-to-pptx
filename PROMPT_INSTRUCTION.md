# 🤖 AI 教案生成指令

將以下指令**完整複製**，貼到 ChatGPT、Claude 或其他 AI 助手，即可自動生成符合 `教案轉 PPTX` 格式的教案。

---

## 📋 複製以下內容給 AI：

```
你是一個專業的教案撰寫助手。請根據以下要求，生成一份結構化的教案，**必須嚴格使用以下 Markdown 格式**：

# 教學主題：[在此填寫主題名稱]

## 基本資訊
- **科目**：[科目名稱]
- **年級**：[年級]
- **時間**：[分鐘數] 分鐘
- **課題**：[課題名稱]

## 教學目標
1. [目標一]
2. [目標二]
3. [目標三]

## 教學資源
- [資源一]
- [資源二]
- [資源三]

## 教學流程

### [導入活動名稱]（[分鐘] 分鐘）
[詳細描述導入活動的內容、教師行為和學生行為]

### [發展活動名稱]（[分鐘] 分鐘）
[詳細描述發展活動的內容、教師行為和學生行為]

### [總結活動名稱]（[分鐘] 分鐘）
[詳細描述總結活動的內容]

## 評估方法
- [評估方式一]
- [評估方式二]

---

格式規則：
- 必須使用 `#` 作為標題、`##` 作為區段標題、`###` 作為教學流程的子步驟
- 每個教學流程步驟必須包含時間
- 使用 `- **key**：value` 格式表示基本資訊
- 使用數字列表 `1. ` 表示教學目標
- 使用 `- ` 表示一般列表項
- 不要加入任何其他格式或多餘的說明文字
```

---

## 📝 使用方式

1. **複製** 上面框框內的完整指令
2. **貼到** ChatGPT / Claude / Gemini 等 AI 工具
3. **補充** 你想要的教學主題（例如：「請生成一個關於光合作用的生物科教案」）
4. AI 會輸出符合格式的教案內容
5. 將輸出**貼到** [教案轉 PPTX 網站](https://lesson-plan-to-pptx.vercel.app)，即可一鍵下載精美 PowerPoint

---

## ✅ 格式檢查清單

生成後請確認教案包含以下元素：

- [ ] 第一行以 `# ` 開頭（教學主題）
- [ ] 有 `## 基本資訊` 區段，包含 `- **科目**：`、`- **年級**：`、`- **時間**：`
- [ ] 有 `## 教學目標` 區段，使用 `1. `、`2. `、`3. ` 列出
- [ ] 有 `## 教學資源` 區段
- [ ] 有 `## 教學流程` 區段，其中的步驟用 `### ` 開頭，並包含時間
- [ ] 有 `## 評估方法` 區段
- [ ] 沒有額外的說明文字或格式標記

## 🌟 範例輸出

以下是 AI 應該生成的格式範例：

```
# 教學主題：光合作用

## 基本資訊
- **科目**：生物科
- **年級**：中三
- **時間**：40 分鐘
- **課題**：光合作用的過程與重要性

## 教學目標
1. 說明光合作用的定義及化學方程式
2. 辨識光合作用所需的條件與產物
3. 理解光合作用對生態系統的重要性

## 教學資源
- 教科書第四章
- 光合作用動畫影片
- 實驗器材（碘液、酒精、本生燈）
- 工作紙

## 教學流程

### 導入活動（5 分鐘）
播放一段植物生長的快鏡影片，提問：「植物為什麼不需要吃東西也能長大？」引導學生思考植物如何獲取能量。

### 發展活動（25 分鐘）
教師講解光合作用的基本概念與化學方程式（10 分鐘），然後進行光合作用實驗示範（10 分鐘），最後小組討論光合作用對生態系統的影響（5 分鐘）。

### 總結活動（10 分鐘）
學生完成工作紙，鞏固本課所學。教師選取 2-3 位學生分享答案並給予回饋。

## 評估方法
- 課堂問答表現
- 實驗觀察記錄
- 工作紙完成準確度
```

---

## ⚠️ 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 使用 `**標題**` 而非 `# 標題` | 第一行必須用 `# ` 開頭 |
| 教學流程沒有用 `### ` 分步驟 | 每個步驟必須用 `### ` 標示 |
| 步驟沒有標明時間 | 格式：`### 活動名稱（X 分鐘）` |
| 基本資訊格式不一致 | 統一使用 `- **key**：value` |
| 加入了多餘的 markdown 格式 | 只使用指定的 `#`、`##`、`###`、`- `、`1. ` |
