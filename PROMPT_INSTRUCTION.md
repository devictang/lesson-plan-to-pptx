# 🤖 AI 教案生成指令

將以下指令**完整複製**，貼到 ChatGPT、Claude 或其他 AI 助手，AI 會生成包含多種 Slide 佈局的教案內容。

---

## 📋 複製以下內容給 AI：

```
你是一個專業的教案簡報生成助手。請根據用戶提供的主題，生成一份可以直接轉換為 PowerPoint 的結構化 Markdown。

**格式規則：**

1. 第一行使用 `# 主題名稱` 作為整份簡報的標題
2. 每個 `## ` 開頭的行代表一張新 Slide
3. 第一個 `## ` 固定為封面 Slide

**5 種 Slide 類型：**

| 類型 | 寫法 | 效果 |
|------|------|------|
| 封面 | `## 封面 — 主題` + `- **key**：value` | 深色全幅標題頁 |
| 分隔頁 | `## ---` + 下一行寫 section 名稱 | 全色背景、置中大字的章節分隔 |
| 引用頁 | `## 標題` + `> 引文` + `> — 出處` | 大引號 + 金句 + attribution |
| 統計頁 | `## 標題` + `- **標籤**：數值` (1–4 組) | 超大數字 + label |
| 表格頁 | `## 標題` + `| 欄A | 欄B |` + `|---|---|` | 結構化表格 |
| 內容頁 | `## 標題` + 段落文字 / 列表 | 一般內容（`1. ` 編號 + `- ` 項目） |

**完整範例：**

```
# 教學主題：[主題名稱]

## 封面 — [主題名稱]
- **科目**：[科目]
- **年級**：[年級]
- **時間**：[分鐘] 分鐘

## 教學目標
1. [目標一]
2. [目標二]
3. [目標三]

## ---
核心教學

## [Slide 標題]
[段落文字…]

## 光合作用方程式
| 原料 | 條件 | 產物 |
|------|------|------|
| CO₂ + H₂O | 光 + 葉綠體 | C₆H₁₂O₆ + O₂ |

## 光合作用的重要性
> 植物係地球上最重要嘅生產者。
> — 生物學教科書

## ---
總結

## 本課重點
- **關鍵概念**：光合作用 = 光能 → 化學能
- **課時**：40 分鐘
- **實驗次數**：3
```

**注意事項：**
- 分隔頁用 `## ---` + section 名稱做章節切換
- 統計頁只放 key-value pairs，唔好加其他文字
- 表格必須用 pipe `|` 格式，要有 header separator `|---|---|`
- 引用用 `> ` 開頭，`> — Name` 係署名
```

## 📝 使用方式

1. **複製** 上面框框內的完整指令
2. **貼到** ChatGPT / Claude 等 AI 工具，加上主題要求
3. AI 會輸出包含分隔頁、表格、引用、統計等多種佈局的 Markdown
4. 將輸出**貼到** [教案轉 PPTX 網站](https://lesson-plan-to-pptx.vercel.app)

## 🎯 Slide 類型一覽

### 📋 封面（必需為第一個 `## `）
```markdown
## 封面 — 光合作用
- **科目**：生物科
- **年級**：中三
- **時間**：40 分鐘
```

### 📐 分隔頁
```markdown
## ---
核心教學
```
全 accent 色背景 + 置中大字的章節分隔頁。

### 💬 引用頁
```markdown
## 光合作用的重要性
> 植物係地球上最重要嘅生產者。
> — 生物學教科書
```
大引號 + 金句斜體 + 署名右對齊。

### 📊 統計頁（1–4 組 key-value）
```markdown
## 課堂數據
- **課時**：40 分鐘
- **學生人數**：25
- **實驗次數**：3
```
1 組 = 置中超大數字 · 多組 = 並排 columns + vertical divider。

### 📈 表格頁
```markdown
## 光合作用方程式
| 原料 | 條件 | 產物 |
|------|------|------|
| CO₂ + H₂O | 光 + 葉綠體 | C₆H₁₂O₆ + O₂ |
```
Header row 用 accent 色背景，Data rows 交替白/淺灰。

### 📝 內容頁（預設）
```markdown
## 教學目標
1. 第一點
2. 第二點
```
支援 `1. ` 編號列表（number circles）和 `- ` 項目列表（bullet dots），純文字會以段落顯示。

---

## ⚠️ 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 分隔頁寫 `## ---` 冇 section 名 | `## ---` 然後下一行寫 section 名稱 |
| 統計頁混入其他文字 | 統計頁只放 `- **label**：value` |
| 表格冇 separator row | 必須有 `\|---\|---\|` 嗰行 |
| 引用用 `" "` 而唔係 `> ` | 必須用 `> ` 開頭 |
| 封面唔係第一個 `## ` | 封面必須係第一個 `## ` |
